# Sales-Performance-Customer-Insights-Dashboard
This project analyzes retail sales data to uncover sales trends, regional performance, product category contribution, and customer segmentation using Python, SQL, and Power BI.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- MS Excel

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Time-based sales trend analysis
- Regional and category-wise performance analysis
- Customer segment analysis

## Power BI Dashboard
The interactive Power BI dashboard includes:
- KPI cards (Total Sales, Total Orders, Avg Sales per Order)
- Monthly sales trend visualization
- Region-wise sales comparison
- Category-wise sales distribution

## Key Insights
- Sales exhibit seasonal trends with noticeable peaks in specific months.
- Certain regions contribute a significantly higher share of total sales.
- Technology category drives the highest sales among product categories.
- Consumer segment dominates overall sales contribution.

## Repository Structure
- data/ - Dataset used
- notebooks/ - Python EDA notebook
- sql/ - SQL analysis queries
- dashboard/ - Power BI dashboard and screenshots

## How to Use
- Open the `.pbix` file in Power BI Desktop to explore the interactive dashboard.
- Review the Jupyter notebook for data preprocessing and EDA.
- Refer to SQL scripts for analytical queries.
